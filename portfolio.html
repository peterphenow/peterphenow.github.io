Realm Roller

This was my first attempt at a group project with my bootcamp cohort. We were tasked to use 2 server-side APIs, find and use a new CSS framework other than Bootstrap, and create a responsive and polished website. My group and I decided to create a 1-stop shop for everything a D&D Master might need to run a successful campaign.

Weather Dashboard

This weather dashboard was my first assignment using an API with an AJAX call and response. In order to retrieve all the necessary information, I actually needed to complete 2 AJAX calls, with the second call dependant on the data received from the first.

Workday Scheduler

This workday scheduler was my introduction to the moment.js library to work with dates and times. The user is able to enter and save notes for each hour of a work day. Throughout the workday, each time slot is dynamically updated to show past, present, and future hours as different colors.
